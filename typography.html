Typography

We have selected “Open Sans” for our app typography because it is a very flexible font and is readable. It also fits our target audience since the font looks more casual and user-friendly. In addition, Open Sans has an extensive font variation in terms of font weights so that we can use it around in different text areas of our app. We will only use one font for uniformity and to fit our style for minimalism. Below are examples of how are we going to use it:

  • Header Titles- Extra Bold font-weight, h1 tag
  • Heading- Bold font-weight, h2 tag
  • Button Titles / Tip Titles- Semi-bold font-weight, font size 24
  • Description / Filter Descriptions- Regular font-weight, font size 20
